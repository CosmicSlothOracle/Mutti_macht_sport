# Bundesliga Live-Ergebnisse

Ein Portal für die Ergebnisse der 1. Fußball-Bundesliga. Spieltage anzeigen, Torschützen analysieren und Statistiken als PDF exportieren.

## Datenquelle

Daten werden von der kostenlosen [OpenLigaDB](https://www.openligadb.de/) API geladen – keine API-Keys erforderlich.

## Lokal starten

**Voraussetzung:** Node.js

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Deployment auf GitHub Pages erfolgt automatisch bei Push auf `main`.
